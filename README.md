## Usage

1. First set everything right of config file
2. Set socket port to 8885 in Cadence
2. First time run:
   make init
3. For LVS:
   make lvs-all TOP_MODULE=XX
4. For DRC:
   make drc-all TOP_MODULE=XX
5. For ALL:
   make all TOP_MODULE=XX
